# travel-assistant-challenge- Frontend

## Main Libs used

- ReactStrap
- react router

## Hosted [[Here](https://travel-assistant-challenge.vercel.app/)]

- By Vercel.app 

## Start with

1. npm install
2. Setup all enviroment variables on env.tsx
   - BACKEND for backend link
   - OPEN_WEATHER for the forecast
   - NINJA_API for currency conversion and country statistics
4. run npm run dev
